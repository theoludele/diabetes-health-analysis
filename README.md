.Diabetes Health Analysis

This project explores health indicators associated with diabetes using a large real-world dataset.

.Objective
To analyze health and lifestyle factors related to diabetes and identify significant associations using statistical methods.

.Dataset
The dataset contains over 200,000 records with health-related features such as blood pressure, cholesterol levels, BMI, physical activity, and lifestyle habits.

.Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Jupyter Notebook

.Key Analysis Steps
- Data cleaning and removal of duplicates
- Feature transformation for better interpretability
- Exploratory data analysis (EDA)
- Chi-square tests for association
- BMI categorization and analysis

.Key Insights
- A strong association was found between diabetes and high blood pressure
- BMI showed a significant relationship with cardiovascular risk factors
- Some variables showed no significant association, highlighting data-driven decision making
